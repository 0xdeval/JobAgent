# Job Hunting Advisor

This chat helps you inspect your active job pipeline using the files generated by your automation flows.

## What this advisor can do

- Show which jobs you already applied to
- Summarize current application status and fit scores
- Retrieve vacancy details for specific roles
- Point you to generated artifacts (CV, cover letter, QA answers)
- Explain gaps or missing files in your pipeline

## Example prompts

- `Which jobs have I already applied to?`
- `Give me the status of all applications from the latest run.`
- `Show me jobs with highest fit score.`
- `Do I already have a cover letter for Chainlink product manager role?`
- `What are the missing artifacts for each application folder?`

## Notes

- Answers are based on current local files.
- If expected files are missing, the advisor will tell you what could not be found.
